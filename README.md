# Smart Campus Placement Management System

## Overview
A web-based placement management system developed for students and placement officers to streamline campus recruitment activities.

## Features
- Student Registration & Login
- Placement Officer Dashboard
- Company Management
- Eligibility Checking (CGPA & Branch)
- Application Tracking
- Responsive User Interface

## Tech Stack
- React.js
- JavaScript
- HTML5
- CSS3
- Firebase / Local Storage

## Modules
### Student Module
- View Companies
- Apply for Companies
- Track Application Status

### Admin Module
- Add/Edit/Delete Companies
- Manage Applications
- Placement Analytics

## Future Enhancements
- Resume Upload
- Email Notifications
- Interview Scheduling
